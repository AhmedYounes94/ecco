Visualization tools for NLP machine learning models.

Installation
============

::

    pip install ecco

You can also install the in-development version with::

    pip install https://github.com/jalammar/ecco/archive/master.zip


Documentation
=============


To use the project:

.. code-block:: python

    import ecco
    ecco.longest()
